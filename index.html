Blue Advantage

Blue Advantage only exists in BLITZ when teams have an uneven number of players.

In BLITZ, when a team is defending, each player has 5 lives. If one team has fewer players, that team also has fewer total lives, which can create an unfair disadvantage.

To help balance the match, the team with fewer players is always assigned to Blue Team. This team may receive a small skill advantage so it can better compete despite having fewer players and fewer total lives.

Because of this rule, the generator only shows matchups where Blue Team has the advantage. Matchups where Red Team would have the advantage are excluded, since Red already has more players and showing those combinations would defeat the purpose of balancing the teams.

Blue Advantage is only applied when using the Difference 1 or higher filter. Selecting the Difference 0 will display every possible matchup without applying this balancing rule. Even if one team has fewer players.

The goal is to keep matches competitive, fair, and balanced even when teams do not have the same number of players.
